The Basics of the Web
How the Web Works

The computer uses a web browser to connect to the web which is made up of HTML, urls, https, and web applications. The browser then connects to the Internet, which is the world's largest computer network, and reads HTML documents that are living in a server on another computer.

Here is the video that breaks it down: Components Of The Web

HTML

A web page is a text document written in a language called HTML which stands for hypertext markup language and is the main protocol of the web. HTML is made up of: text content, markup (is what it looks like), references to other documents (such as images and videos), and links to other pages

Tags and Elements

An HTML tag is a command within angled brackets that tell the computer what to do. There are two types of tags, an open tag and closing tag. An element refer to anything within an open and closing tag. While an attribute is a property of an HTML element, it used in the placement of a url link for a video, for example.

Why Computers are Stupid

Computers won't be able to figure out what a programmer intends it to do on its’ own. So that is the reason why it is very important for a programmer to write carefully what they want a computer to do. In other words, write with correct syntax. Making sure a code is correct eliminates errors.

Inline vs Block Elements

HTML elements can be either block elements or inline elements. Block elements create an imaginary block surrounding the content you have in between. Inline elements allow the content you have between them to be inline with the rest of the content that precedes it or follows. An example of a block element is a div and of a inline element is a span.

HTML's "Tree-Like" Structure

HTML elements can have other elements within them. In order to view the elements easily in a simplistic fashion they are set up into nested boxes that create a tree-like structure. This structure is what programmers call DOM (Document Object Model).

Here is a video that mentions DOM: How HTML, CSS and DOM Are Related

Indentations and Boxes

The HTML elements in the tree-like structure are indented if they fall within a nested box. The indentation will move closer to the right as it falls further and further in the code.
